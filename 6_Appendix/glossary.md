# 用語集

## 略語・定義

### A

- **API**: Application Programming Interface（アプリケーションプログラミングインターフェース）
- **APM**: Application Performance Monitoring（アプリケーションパフォーマンスモニタリング）

### C

- **CDN**: Content Delivery Network（コンテンツ配信ネットワーク）
- **CI/CD**: Continuous Integration / Continuous Deployment（継続的インテグレーション・継続的デプロイメント）

### D

- **DR**: Disaster Recovery（災害復旧）

### E

- **ERD**: Entity Relationship Diagram（エンティティ関係図）

### G

- **GDPR**: General Data Protection Regulation（一般データ保護規則）

### J

- **JWT**: JSON Web Token（JSONウェブトークン）

### M

- **MVP**: Minimum Viable Product（最小実行可能製品）

### N

- **NFR**: Non-Functional Requirement（非機能要件）

### P

- **POC**: Proof of Concept（概念実証）

### R

- **RACI**: Responsible, Accountable, Consulted, Informed（責任分担マトリックス）

### S

- **SPA**: Single Page Application（シングルページアプリケーション）

### T

- **TLS**: Transport Layer Security（トランスポート層セキュリティ）

### U

- **UI**: User Interface（ユーザーインターフェース）
- **UX**: User Experience（ユーザーエクスペリエンス）

### W

- **WBS**: Work Breakdown Structure（作業分解構造）
- **WCAG**: Web Content Accessibility Guidelines（ウェブコンテンツアクセシビリティガイドライン）
